Maintenance Mode

& Emergency Broadcast Settings

Maintenance Mode

{{InputCSRF}}

This page allows you to set various Maintenance Mode flags that can pause or disable website features if needed. In an emergency, click on the Lock Down Everything button that will enable ALL maintenance mode flags and basically disable the whole website for everybody except admin user accounts.

Custom Messaging

You can include a custom headline and message to be shown while Maintenance Mode is activated (or included at the top of every page of the site even when Maintenance Mode is not active, by selecting the appropriate option below).

For example, if the Signup or Login page are disabled and the Maintenance Mode page is dispayed, the custom messaging will appear on only those pages.

The custom messaging will appear on top of every page of the site. This is useful to announce upcoming maintenance work ahead of time, e.g. before Maintenance Mode is actually activated.

Maintenance Mode Settings

New account signups are paused and a maintenance page is shown in its place.

The login page is disabled (except for admin user login). Already logged-in users can remain logged in.

No new entries into the chat room are allowed.

Every site feature becomes basically 'admin required' and users are given an error page where their only option is to log out, delete their account, or come back later.

Emergency Kill Switch

{{InputCSRF}}

This section is reserved for major emergencies, such as the sudden disappearance of the website's owner. This special use case of Maintenance Mode is driven by a "dead man's switch" which will automatically activate if the site's owner has not logged in for an extended period of time.

When the Emergency Kill Switch is activated, Maintenance Mode will be enabled automatically with the following settings:
  • New signups will be disabled
  • New logins for existing users will continue to work
  • All 'login required' pages will display the Maintenance Mode messaging, and give existing users the ability to fully delete their account, log out, or try coming back later.
  • All other site features (chat, forums, etc.) will be locked down to admin-only.

Dead Man's Switch

{{if .SiteOwner}} {{.SiteOwner.Username}} Last seen: {{SincePrettyCoarse .SiteOwner.LastLoginAt}} ago {{else}} not configured! {{end}}

The number of days since the site owner's last login before Emergency Mode activates. Enter 0 (zero) to disable the dead man's switch.

If this box is checked, and the Site Owner and Days Missing values are filled in (non-zero), then the Emergency Mode will activate when the site owner has gone missing.

This checkbox will force Emergency Mode to 'Activated.' It can be used for testing the feature, or, in case the feature were automatically activated due to the owner's absence, the feature can be disarmed by unchecking this box.
